##  Функциональность

###  Аутентификация и пользователи
- Регистрация с подтверждением email
- JWT-аутентификация
- Восстановление пароля через email
- Защищенные маршруты с middleware
- Ролевая модель (user, admin, developer)

###  Каталог товаров
- Список аккумуляторов с пагинацией
- Детальные характеристики каждого аккумулятора
- Фильтрация по параметрам (в разработке)
- Предустановленный набор из 42 реальных аккумуляторов

###  Корзина покупок
- Добавление/удаление товаров
- Изменение количества
- Автоматическое суммирование
- Защита от дубликатов

###  Избранное
- Добавление в избранное
- Удаление из избранного
- Просмотр списка избранного

###  Заказы
- Оформление заказа из корзины
- Транзакционная целостность
- Квитанция на email с деталями заказа
- История всех заказов пользователя

###  Email-уведомления
- Подтверждение регистрации
- Код сброса пароля
- Квитанция о заказе
- Поддержка SMTP (Mailtrap для тестов)

###  Документация
- Swagger UI с полным описанием API
- README с инструкциями
- Docker-контейнеризация

##  Быстрый старт

### Предварительные требования

- **Go** 1.23 или выше
- **Docker** и **Docker Compose** (для production-запуска)
- **PostgreSQL** 15 (для локальной разработки без Docker)
- **Make** (опционально, для удобства)
